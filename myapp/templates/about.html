DESTRUCTION ANALYSIS
  Using Machine Learning

    Our project focuses on using machine learning to analyze satellite images for identifying
damaged and undamaged buildings. By leveraging advanced algorithms, we aim to create a robust
system that can accurately classify buildings in post-disaster scenarios.

    Using high-resolution satellite datasets to build predictive models to help vulnerable coastal
communities adapt to evolving conditions and recover from extreme climate events. This will result in
new and innovative approaches designed to increase the impact of response activities.
